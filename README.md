# 关键词
Lucene搜索引擎 快速查词 全文搜索 页面翻译 笔记书签 离线词典 在线词典  

可打开 PDF MDX DSL.DZ 等文件，作为词库查询，或作为书籍阅读。  

具体特性可查看[更新日志](PLOD/src/main/assets/rizhi.md)、[旧自述](README_anteque.md)。
